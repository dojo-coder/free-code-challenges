# Contributing to Free Code Challenges

Thanks for your interest in contributing! This list is maintained by the [DojoCode](https://dojocode.io) community.

## How to Contribute

### Suggest a Challenge

1. [Create a challenge on DojoCode](https://dojocode.io)
2. Open an issue using the **"Suggest Challenge"** template
3. Include: title, difficulty, category, and your DojoCode challenge URL

### Add a Challenge to This List

1. Fork this repository
2. Add your challenge in the appropriate category section
3. Follow the entry format below
4. Submit a pull request

### Report a Broken Link

Use the **"Report Broken Link"** issue template to let us know about any links that no longer work.

## Entry Format

Each challenge entry should follow this format:

```markdown
- [Challenge Title](https://dojocode.io/challenge/username/slug/template_key) — Brief one-sentence description. `tag1` `tag2`
```

**Example:**

```markdown
- [Binary Search](https://dojocode.io/challenge/daniku/binary-search/nodejs_jest) — Search for a target in a sorted array using binary search. `searching` `divide-and-conquer`
```

## Guidelines

- Challenges must be **free** and publicly accessible on [DojoCode](https://dojocode.io)
- Descriptions should be **concise** — one sentence that explains the problem
- Place challenges in the **correct category** and difficulty level
- Use **lowercase backtick tags** for related concepts
- Keep entries in order: beginner first, expert last within each section
